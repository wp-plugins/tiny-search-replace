== Description ==

The Tiny Search Replace plugin re-adds search and replace support to the TinyMCE Rich Visual Editor
The TinyMCE searchreplace plugin is normally bundled with TinyMCE, but it has be removed from the
version within WordPress.  This plugin puts it back in.

Most of the credit for this plugin belongs to [Moxiecode Systems] (http://tinymce.moxiecode.com/ ),
who wrote the searchreplace plugin for TinyMCE, which this plugin uses.

== Installation ==

1. copy the 'tiny-search-replace' directory to your 'wp-contents/plugins' directory.
1. Activate the Tiny Search Replace plugin in your plugins administration page.
1. You will now see a search and a replace button when using the the TinyMCE Rich Visual Editor.
